
##  Key Files  
- **Global Weather Repository .csv**: Contains daily weather metrics (temperature, humidity, etc) for many cities/countries globally.  
- **temprature.ipynb**: Notebook performing steps such as:  
  - Loading the dataset with e.g. `pandas`.  
  - Data cleaning (missing values, inconsistent formats).  
  - Exploratory visualisations (temperature trends over time, across geographies).  
  - Summary statistics and possibly time-series decomposition or correlation analysis.  
- **requirements.txt**: A list of Python packages required, for example `pandas`, `numpy`, `matplotlib`, `seaborn`, `jupyter`, etc.
